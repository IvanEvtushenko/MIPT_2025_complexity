## Дерево Штейнера

В настоящем репозитории представлен исходный код проекта "Дерево Штейнера" по предмету "Сложности вычислений", выполнены Евтушенко Иваном Андреевичем, студентом МФТИ ФПМИ 3 курса группы Б05-320 в осеннем семестре 2025г.

Проект выполнен с использованием chat-GPT для поиска информации и написания кода для тестирования алгоритма. Остальнеы части работы выполнены Иваном самостоятельно с использованием указанных в .pdf источников

Для воспроизведения тестов необходимо скачать проект и запустить .ipynb. Тестирование запускается в многопоточном режиме, ибо необходимый переборный алгоритм работает долго.
